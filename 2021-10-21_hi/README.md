## requirements

```
$ love --version
LOVE 11.3 (Mysterious Mysteries)
```

## to run

put the `hi.mp3` ([this song](https://soundcloud.com/sin_clav/hi-1)) into this project root and type

```
$ love .
```

and press `a` key to start music.
